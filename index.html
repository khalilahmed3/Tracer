Il problema è risolto?

Inizializzazione Hoya GT-3000

  • SPEGNETE e RIACCENDETE il Tracciatore.
  • Assicuratevi che sia in AUTO MEASURE <START>.

Hoya: Configurazione Porta (H100)

TENETE PREMUTO SELECT e PREMETE 3 VOLTE ITEM. Poi PREMETE START per il MENU CONFIGURAZIONE.

Cercare H100 SET PORT e confermare con START.

Hoya: Abilitazione Barcode

Con il tasto SELECT impostare l’opzione BARCODE. Per abilitarlo alla funzione BARCODE>BARCODE premete i tasti UP & DOWN, poi CONFERMATE con START.

Hoya: Parametri Comunicazione (H101)

PREMETE RESET per USCIRE dal menu. Poi cercate H101 SET COMM e confermate con START. Dovrebbero esserci le impostazioni 9600 N 1.

Hoya: Impostazione Protocollo (H103)

PREMETE RESET per USCIRE dal menu. Poi cercate H103 SET PROTOCOL e confermate con START. Selezionare DATA OMA.

Hoya: Verifica Parametri OMA (H104)

Cercare H104 SET OMA e confermare con START. Verificare i parametri OMA.

Hoya: Salvataggio e Test Finale

Punti da CORREGGERE: POINT: 400, CIRC: ON, HBOX: ON, VBOX: ON. SPEGNETE e RIACCENDETE il Tracciatore per COMPLETARE la procedura.

Esecuzione Service Restart

Ricercare il comando SYSTECH SERVICE RESTART ed avviarlo con Run This Command.

Il problema è risolto?

Protocollo Essilor (Combinazione Tasti)

Selezionare contemporaneamente Tasto 1 e Tasto VERDE SPUNTATO su ESSILOR TESS.

Essilor: Menu Communicate

Premere Tasto 6 (COMMUNICATE). Confermare con VERDE.

Essilor: Menu Protocol

Premere Tasto 1 (PROTOCOL). Confermare con VERDE.

Essilor: Verifica Videata Protocollo

Possibile che si possa presentare la videata seguente:

Essilor: Selezione Pc OmaRs232

Premere Tasto 1 (PC OmaETH). Confermare con VERDE. Ricercare Pc OmaRs232 e confermare.

Essilor: Configurazione Baud Rate

Tasto ROSSO per uscire. Poi COMMUNICATE -> PARAMETERS -> PC PORT. Baud: 9600 (HOYA) o 19200 (TESS).

Il problema è risolto?

Impostazioni Moxa Avanzate

Verificare impostazioni moxa su Personality Server (IP Systech).

Moxa: Basic & Network Settings

Assicurarsi che BASIC SETTINGS e NETWORK SETTINGS siano settati correttamente.

Moxa: Serial Port Settings

SERIAL SETTINGS > PORT 1: Baud rate HOYA 9600, TESS 19200. OPERATING SETTINGS: PORT 1.

Moxa: Operating Settings

Assicurarsi che il menu OPERATING SETTINGS > PORT 1 sia settato come da immagine. Confermare poi con SUBMIT.
